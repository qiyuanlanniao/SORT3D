## Overview

To address the issues of missing spatial reasoning and context explosion in large-scale 3D scenes using LLM, an omnidirectional perception-driven embodied intelligence agent framework was developed based on Python, PyTorch, and ROS2. By constructing a dynamic scene graph as the agent’s world model and combining it with a multi-resolution spatial attention mechanism, complex long-range navigation in zero-shot environments was achieved with a success rate of 93.18%, while reducing token costs by up to 69.98%.
